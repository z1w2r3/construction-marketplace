# NotebookLM 深度研究

对特定主题进行深度研究，构建知识图谱。

---

**研究主题**: $ARGUMENTS

---

## 核心原则 ⭐ 新增

在整个执行过程中,我将遵循以下原则:
1. **透明化思考**: 每个决策都说明理由
2. **真实优先**: 绝不编造数据,所有信息必须有来源
3. **用户确认**: 重要决策前等待用户确认
4. **信息溯源**: 所有引用必须标注文档来源

---

## 执行流程

### 步骤 0: 研究范围规划 ⭐ 新增

💭 **我的研究策略**:
```
📋 研究主题: "$ARGUMENTS"

🎯 研究目标分析:
- 这是 [技术研究/业务分析/历史追溯/对比研究]
- 需要关注的维度: [维度1, 维度2, 维度3]
- 预计深度: [概览/中等/深入]

📊 检索策略:
- 关键词: [列表]
- 优先文档类型: [技术文档/业务文档/会议记录]
- 预计阅读文档数: [范围]

⏱️  预计耗时: [估计]
```

询问用户确认或调整:
```
我计划对 "$ARGUMENTS" 进行 [深度] 研究,预计阅读 [N] 个文档。

您是否想要:
1. 调整研究范围或深度?
2. 指定特定的文档或时间范围?
3. 继续按当前策略执行?

(直接回复"继续"即可开始,或告诉我需要调整的地方)
```

---

### 步骤 1: 检索相关文档

💭 **检索思考过程**:
- 使用关键词: [列表]
- 检索范围: [全库/特定目录]
- 排序方式: [相关度/时间]

使用 **Skill tool** 调用 `smart-retrieval` skill:
- query: $ARGUMENTS
- top_k: 10（深度研究需要更多文档）

### 2. 深度阅读文档

使用 **Skill tool** 调用对应的 document skills (docx/pdf) 进行深度阅读。

对每个文档:
- 提取核心观点
- 识别关键数据
- 构建主题关联

### 步骤 3: 综合分析

💭 **分析思路**:
```
📊 已读取 [N] 个文档,现在进行综合分析:
- 共同主题: [主题列表]
- 观点分布: [一致/分歧/互补]
- 信息完整度: [完整/部分缺失/大量缺失]
```

进行多文档综合分析:
- analysis_type: "pattern"（模式发现）
- 识别共同主题
- 发现隐含关系
- 构建知识网络

---

### 步骤 3.5: 缺失信息检测 ⭐ 新增 (来自文章启发)

检查研究主题所需的关键信息是否齐全:

**如果发现信息缺口**,创建协作文档:

文件路径: `.notebooklm/sessions/待补充信息-[日期时间].md`

内容模板:
```markdown
# 待补充信息清单

## 研究主题
$ARGUMENTS

## 已找到的文档
[自动生成列表]

## 缺失的关键信息
□ 信息1: [说明为什么需要]
  - 建议补充文档类型: [具体说明]
  - 重要程度: [高/中/低]

□ 信息2: [说明为什么需要]
  - 建议补充文档类型: [具体说明]
  - 重要程度: [高/中/低]

## 替代方案
如果无法补充文档,我可以:
- [ ] 基于现有信息生成部分报告(明确标注信息缺口)
- [ ] 调整研究范围,聚焦于有充分信息的部分
- [ ] 使用外部知识(WebSearch)补充(需您授权)

## 下一步
补充完成后,请回复"继续研究",我将基于完整信息重新生成报告。
```

输出提示:
```
⚠️  发现信息缺口,已创建协作文档:
   .notebooklm/sessions/待补充信息-YYYYMMDD-HHMMSS.md

您可以:
1. 补充缺失的文档后回复"继续研究"
2. 回复"基于现有信息继续"(报告中会标注缺失部分)
3. 回复"调整研究范围"(告诉我聚焦哪些方面)
```

**如果信息完整**,继续下一步。

---

### 步骤 4: 生成研究报告

💭 **报告构建思路**:
```
📝 报告结构:
1. 研究范围说明
2. 核心发现 (基于 [N] 个文档)
3. 详细分析 (分 [N] 个维度)
4. 知识图谱
5. 洞察与建议

⚠️  特别注意:
- 所有结论必须有文档支撑
- 缺失信息明确标注
- 多个来源有矛盾时,列出所有观点
```

输出结构化研究报告（Markdown 格式）:

```markdown
# [主题] 深度研究报告

## 📚 研究范围
- 文档数: [N]
- 时间跨度: [起止时间]
- 主要来源: [文档类型]

## 🔍 核心发现
1. [主要发现1]
2. [主要发现2]
3. [主要发现3]

## 📊 关键数据
[统计数据和趋势]

## 🌐 知识图谱
- 主题A → 关联 → 主题B
- 观点X ⇄ 支持/矛盾 ⇄ 观点Y

## 📖 详细分析
[分章节的深入分析]

## 💡 洞察与建议
[基于研究的洞察]

## 📚 参考文献
[所有使用的文档]
```
